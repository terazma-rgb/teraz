# 프로젝트 개요

"서재의 주식 계산기"는 미국 주식과 한국 주식 투자자들이 추가 매수 시 평균 구매 단가를 쉽게 계산할 수 있도록 돕는 웹 애플리케이션입니다. 전문적이고 모던한 서재 책상 분위기의 디자인을 통해 사용자에게 신뢰감을 주는 사용자 경험을 제공합니다.

# 디자인 및 기능 명세

## 디자인 컨셉

- **테마**: 고급 서재의 책상
- **색상 팔레트**: 따뜻한 나무 색상, 짙은 갈색, 베이지, 그리고 강조 색상으로 금색 또는 짙은 녹색을 사용하여 전문적이고 차분한 분위기를 연출합니다.
- **폰트**: 가독성이 좋으면서도 클래식한 느낌을 주는 'Pretendard' 폰트를 사용합니다.
- **레이아웃**:
    - 전체 화면을 차지하는 배경에 나무 질감의 이미지를 사용합니다.
    - 계산기 인터페이스는 카드 형태로 디자인하여 입체감을 줍니다.
    - 반응형 디자인을 적용하여 모바일 환경에서도 최적화된 화면을 제공합니다.

## 기능

- **통화 선택**: 미국 주식(USD)과 한국 주식(KRW) 계산을 위한 탭 또는 스위치를 제공합니다.
- **입력 필드**:
    - **현재 보유 주식 수**: 사용자가 현재 가지고 있는 주식의 수량
    - **현재 평균 단가**: 현재 보유 주식의 평균 구매 가격
    - **추가 매수 주식 수**: 새로 구매할 주식의 수량
    - **추가 매수 단가**: 새로 구매할 주식의 가격
- **계산 결과**:
    - **총 보유 주식 수**: 계산 후 총 주식 수량
    - **최종 평균 단가**: 추가 매수 후의 새로운 평균 구매 단가
    - **총 투자 금액**: 전체 투자 원금
- **초기화 기능**: 모든 입력 필드를 한 번에 초기화하는 버튼을 제공합니다.

# 기술 스택

- **HTML**: 웹 페이지의 구조를 정의합니다.
- **CSS**: 모던 CSS 기능을 활용하여 디자인과 레이아웃을 구현합니다.
    - Flexbox 및 Grid 레이아웃
    - CSS 변수를 활용한 테마 관리
    - Container Queries를 사용한 반응형 컴포넌트
- **JavaScript**:
    - ES6+ 표준을 따릅니다.
    - Web Components (Custom Elements)를 사용하여 재사용 가능한 UI 컴포넌트를 만듭니다.
    - 동적인 계산 로직 및 UI 상호작용을 처리합니다.

# 구현 계획

1.  **HTML 구조 설계**: `index.html` 파일에 기본 마크업을 작성하고, 계산기를 위한 커스텀 엘리먼트(`<stock-calculator>`)를 추가합니다.
2.  **CSS 스타일링**: `style.css` 파일에 전체적인 테마(배경, 색상, 폰트)와 계산기 UI 스타일을 적용합니다.
3.  **JavaScript 로직 구현**:
    - `main.js` 파일에 `<stock-calculator>` 커스텀 엘리먼트를 정의합니다.
    - Shadow DOM을 사용하여 컴포넌트의 스타일과 구조를 캡슐화합니다.
    - 사용자 입력을 받아 평균 단가를 계산하는 로직을 작성합니다.
    - 통화 변경(USD/KRW) 기능을 구현합니다.
4.  **다듬기**: 반응형 디자인을 세부적으로 조정하고, 사용자 경험을 향상시키기 위한 애니메이션 효과 등을 추가합니다.
