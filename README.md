# 주식 물타기 계산기

## 프로젝트 개요

전문가용 주식 물타기 계산기로, 미국주식과 한국주식의 물타기 전략을 분석하고 시각화합니다. 서재 책장 배경의 전문적인 분위기와 모던한 디자인을 결합한 투자 분석 도구입니다.

## 주요 기능

### ✅ 구현된 기능

1. **이중 통화 지원**
   - 미국주식 (USD, $)
   - 한국주식 (KRW, ₩)
   - 실시간 원/달러 환율 표시 (USD/KRW)

2. **물타기 계산 기능**
   - 현재 보유 수량 및 평균 단가 입력
   - 현재 시가 및 추가 매수 정보 입력
   - 물타기 전후 비교 분석
   - 손절가 계산
   - 목표가 도달 시 예상 수익률

3. **시각화 기능**
   - 수익률 비교 막대 차트
   - 현재 vs 물타기 후 vs 목표 수익률
   - Chart.js를 활용한 인터랙티브 차트

4. **사용자 인터페이스**
   - 탭 기반 네비게이션
   - 반응형 디자인 (모바일/태블릿/데스크탑)
   - 실시간 입력값 검증
   - 부드러운 애니메이션 효과

5. **전문가용 기능**
   - 회복까지 필요한 상승률 계산
   - 총 투자금 계산
   - 손익 개선 분석

## 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **차트 라이브러리**: Chart.js
- **폰트**: Inter, Noto Sans KR
- **아이콘**: Font Awesome
- **배경**: CSS 그라디언트 및 패턴

## 사용 방법

### 미국주식 계산
1. "미국주식" 탭 선택
2. 현재 보유 수량, 평균 단가, 현재 시가 입력
3. 추가 매수 희망 수량과 가격 입력
4. 목표 매도가 입력
5. "계산하기" 버튼 클릭

### 한국주식 계산
1. "한국주식" 탭 선택
2. 동일한 방식으로 정보 입력
3. 계산 결과 확인

## 계산 결과 항목

### 현재 상황
- 총 보유 수량
- 평균 단가
- 현재 손익
- 수익률

### 물타기 후
- 총 보유 수량
- 신규 평균 단가
- 손익 개선
- 손절가

### 목표 달성 시
- 목표가 도달 시 수익
- 목표 수익률
- 총 투자금
- 회복까지 필요한 상승률

## UI 특징

- **배경**: 서재 책장을 연상시키는 전문적인 그라디언트
- **색상**: 모던한 블루 톤과 중립색 조합
- **타이포그래피**: Inter와 Noto Sans KR 폰트
- **애니메이션**: 부드러운 전환 효과
- **반응형**: 모든 기기에서 최적화된 표시

## 주의사항

이 계산기는 투자 참고용 도구입니다. 실제 투자 시에는 다음 사항을 고려하세요:

- 거래 수수료는 계산에 포함되지 않습니다
- 시장 상황은 실시간으로 변동됩니다
- 전문가의 조언을 함께 고려하세요
- 투자에 대한 모든 책임은 투자자 본인에게 있습니다
